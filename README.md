# PXX Background Management System
一个简单的后台管理系统，基于 `SpringBoot`, `MybatisPlus`, `Thymeleaf`, `SpringSecurity` 


另外还有 `Druid`, `Swagger`, `Lombok`

## 环境

- `JDK 17`
- `SpringBoot 2.6.7`
- `MySQL 8`
